Developer API Documentation

Integrate this Authentication logic into any website.

POST /api/setup

Generates a new secret and QR Image. Client must save the secret.

fetch('https://verify-via-otp.vercel.app/api/setup', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ label: "User", issuer: "App" })
})
POST /api/verify

Mathematically validates the 6-digit token against the secret.

fetch('https://verify-via-otp.vercel.app/api/verify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ 
        token: "123456", 
        secret: "SAVED_USER_SECRET" 
    })
})
